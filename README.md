# SQL Bank Transaction Analysis

A small SQL project analyzing a mock banking database with customers, 
accounts, and transactions.

## What I did
- Designed a 3-table relational database (customers, accounts, transactions)
- Wrote SQL queries using JOIN, GROUP BY, and aggregate functions
- Calculated total money movement by category
- Ranked customers by total account balance
